# インタラクション

Cycle Journal の主要なインタラクションパターンです。

## ジェスチャー

### スワイプアクション

日記・タスクのリストアイテムで統一されたスワイプ操作を提供します。

| 方向 | アクション | アイコン | 色 |
|------|-----------|---------|-----|
| 左スワイプ | 編集 | `pencil` | Blue |
| 右スワイプ | 削除 | `trash` | Red |

### タップ操作

| 画面 | 対象 | アクション |
|------|------|-----------|
| タスク一覧 | チェックボックス | タスク完了 → 振り返りフロー開始 |
| タスク一覧 | タスク行 | タスク詳細を Sheet で表示 |
| 日記一覧 | 日記行 | 日記詳細を表示 |
| コーチホーム | セッション行 | セッション詳細を表示 |
| 週間カレンダー | 日付 | その日の日記にジャンプ |

## アニメーション

### タイピングインジケーター

コーチが応答を生成中に表示されます。

```
● ● ●
```

- 3つのドットが順番にバウンス
- 各ドットに 0.15 秒のディレイ
- 無限ループ
- コーチメッセージの位置（左寄せ + アバター）に表示

### 振り返りプログレスバー

4ステップの進行状況を可視化します。

```
Step 1: ━━━━━━━━━━ ──── ──── ────
Step 2: ━━━━━━━━━━ ━━━━━━━━━━ ──── ────
Step 3: ━━━━━━━━━━ ━━━━━━━━━━ ━━━━━━━━━━ ────
Step 4: ━━━━━━━━━━ ━━━━━━━━━━ ━━━━━━━━━━ ━━━━━━━━━━
```

- アクティブなステップまでの部分が塗りつぶされる
- ステップ間の遷移はスムーズなアニメーション

### チャットメッセージ

- 新しいメッセージが追加されると、自動的に最下部へスクロール
- `ScrollViewReader` + `scrollTo` で実装

### 完了セレブレーション

振り返り完了時のお祝い表示。

- セレブレーションアイコンの表示
- メッセージの段階的フェードイン
- 「閉じる」ボタンの遅延表示

## フィードバック

### ローディング状態

| 画面 | 表示 |
|------|------|
| サインイン | `ProgressView`（Sign in 中） |
| コーチチャット | タイピングインジケーター（AI 応答待ち） |
| API 通信 | `ProgressView`（データ取得中） |

### エラー状態

| 種類 | 表示方法 |
|------|---------|
| 認証エラー | サインイン画面にテキスト表示 |
| API エラー | `Alert` ダイアログ |
| ネットワークエラー | フォールバック応答を返す |

### 空状態

コンテンツがない場合に表示します。

| 画面 | メッセージ |
|------|-----------|
| 日記一覧 | 「日記がありません。最初の日記を書いてみましょう。」 |
| タスク一覧 | 「タスクがありません。」 |
| セッション履歴 | 「まだ対話がありません。」 |

SwiftUI の `ContentUnavailableView` を使用しています。

### 確認ダイアログ

破壊的な操作の前に確認を求めます。

| 操作 | ダイアログ |
|------|-----------|
| 日記の削除 | 「この日記を削除しますか？」 |
| タスクの削除 | 「このタスクを削除しますか？」 |
| セッションの終了 | 「このセッションを終了しますか？」 |
| サインアウト | 「サインアウトしますか？」 |
| 全データクリア | 「すべてのデータを削除しますか？（DEBUG）」 |

## キーボード

### テキスト入力

| 画面 | 入力フィールド | 特徴 |
|------|-------------|------|
| 日記作成 | TextEditor | 複数行、自動フォーカス |
| コーチチャット | TextField | 1〜5行可変、送信ボタン |
| タスク作成（タイトル） | TextField | 単一行 |
| タスク作成（説明） | TextEditor | 複数行 |
| 振り返り各ステップ | TextEditor | 複数行 |

### 入力バリデーション

| フィールド | ルール |
|-----------|--------|
| 日記内容 | 空白トリム後に空でないこと |
| コーチメッセージ | 空白トリム後に空でないこと |
| タスクタイトル | 必須、空でないこと |
| タスク説明 | 任意 |
| タスク期限 | 任意、DatePicker で入力 |
| 振り返り各ステップ | 空白トリム後に空でないこと |
