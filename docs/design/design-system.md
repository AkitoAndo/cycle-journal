# デザインシステム

## デザイン原則

1. **穏やかさ** — ジャーナリングに集中できる静かな UI
2. **シンプル** — 必要な情報だけを提示する
3. **温かみ** — AI コーチとの対話が心地よいと感じられる
4. **一貫性** — 全画面で統一されたビジュアル言語

## スペーシング

8pt グリッドシステムを採用しています。

| トークン | 値 | 用途 |
|---------|-----|------|
| xs | 4pt | テキスト間の最小余白 |
| sm | 8pt | アイコンとテキスト間 |
| md | 12pt | コンポーネント内パディング |
| lg | 16pt | 標準パディング、セクション間 |
| xl | 24pt | セクション間の大きな余白 |
| xxl | 32pt | 画面上部の余白 |

## 角丸（Corner Radius）

| サイズ | 値 | 用途 |
|--------|-----|------|
| Small | 8pt | カード、入力フィールド |
| Medium | 12pt | ボタン、タグ |
| Large | 18pt | メッセージバブル |
| Full | 50% | アバター、アイコン背景 |

## シャドウ

## コンポーネント

### メッセージバブル

コーチとユーザーの対話で使用します。

```
┌─────────────────────────────────────────────┐
│                                             │
│                  ┌──────────────────┐        │
│                  │ 今日は仕事で     │ ← ユーザー
│                  │ 疲れました       │   Blue bg
│                  └──────────────────┘   右寄せ
│                                             │
│  ┌─┐ ┌──────────────────┐                   │
│  │🌳│ │ お疲れさまでした。│ ← コーチ
│  └─┘ │ 何が一番エネルギー│   Gray bg
│      │ を使いましたか？  │   左寄せ + アバター
│      └──────────────────┘                   │
│                                             │
│  ┌─────────────────────────────────┐        │
│  │ メッセージを入力...        送信  │        │
│  └─────────────────────────────────┘        │
└─────────────────────────────────────────────┘
```

| プロパティ | ユーザーバブル | コーチバブル |
|-----------|-------------|-------------|
| 背景色 | Primary Blue | Gray 3 |
| テキスト色 | White | Primary Text |
| 配置 | 右寄せ | 左寄せ |
| 角丸 | 18pt | 18pt |
| アバター | なし | 🌳 ツリーアイコン |

### タスクカード

```
┌─────────────────────────────────────┐
│ ○  タスクタイトル                    │
│    タスクの説明テキスト...           │
│    📅 2025/03/20                     │
└─────────────────────────────────────┘
```

| 状態 | チェック | テキスト装飾 | 期限表示 |
|------|---------|-------------|---------|
| 未完了 | ○ | 通常 | 通常（グレー） |
| 完了 | ✓（緑） | 取り消し線 | — |
| 期限超過 | ○ | 通常 | 赤字 |
