# 画面一覧

Cycle Journal の全画面仕様です。

## 画面マップ

| # | 画面名 | ファイル | 説明 |
|---|--------|---------|------|
| 1 | サインイン | `SignInView` | Apple ID ログイン |
| 2 | 日記一覧 | `DiaryListView` | メイン画面、日記の閲覧・管理 |
| 3 | 日記作成・編集 | `JournalComposerView` | 日記の入力 |
| 4 | コーチホーム | `CoachHomeView` | コーチ機能のトップ |
| 5 | コーチチャット | `CoachChatView` | AI コーチとの対話 |
| 6 | セッション履歴 | `SessionHistoryView` | 過去の対話一覧 |
| 7 | タスク一覧 | `TaskListView` | タスクの管理 |
| 8 | タスク作成 | `AddTaskView` | 新規タスクの入力 |
| 9 | タスク詳細 | `TaskDetailView` | タスクの詳細と振り返り |
| 10 | 振り返り | `TaskReflectionView` | 4ステップ振り返りウィザード |
| 11 | 設定 | `SettingsView` | アカウント・通知・データ管理 |


- スワイプアクション: 左→編集、右→削除

- 検索・カレンダーのツールバーボタン

## 6. タスク一覧画面

**ファイル:** `Views/Tasks/TaskListView.swift`

```
┌─────────────────────────────────────┐
│ タスク                              │
├─────────────────────────────────────┤
│ 未完了                              │
│ ┌─────────────────────────────────┐ │
│ │ ○ 朝の散歩を始める             │ │
│ │   毎朝15分歩く         📅 3/20 │ │
│ ├─────────────────────────────────┤ │
│ │ ○ 日記を毎日書く               │ │
│ │   寝る前に3行          📅 3/25 │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 完了                                │
│ ┌─────────────────────────────────┐ │
│ │ ✓ 瞑想を試す                   │ │
│ │   5分間の呼吸瞑想              │ │
│ └─────────────────────────────────┘ │
│                                     │
│                              ┌───┐  │
```

**機能:**
- 未完了 / 完了のセクション分け
- タスク行: チェック・タイトル・説明・期限

- スワイプアクション: 左→編集、右→削除
- 期限超過は赤字表示
- 空状態の表示（タスクがない場合）

```

┌─────────────────────────────────────┐
│ ━━━━━━━━━━ ━━━━━━━━━━ ──── ────    │

│ └─────────────────────────────────┘ │
│               [← 戻る]  [次へ →]   │
└─────────────────────────────────────┘

ステップ 3: 学びの抽出
┌─────────────────────────────────────┐
│ ━━━━━━━━━━ ━━━━━━━━━━ ━━━━━━━ ──── │

│ ┌─────────────────────────────────┐ │

│ └─────────────────────────────────┘ │

ステップ 4: 次への調整
```
