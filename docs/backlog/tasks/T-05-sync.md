## ユーザーストーリー

> ユーザーとして、タスクデータをサーバーに同期して、端末を変えてもタスクを引き継ぎたい。

## 受け入れ条件

- [ ] タスクのCRUDがサーバーに反映される
- [ ] ふりかえりデータもサーバーに同期される
- [ ] オフライン時はローカルに書き込み、オンライン復帰後に同期

## 検討事項

- 同期の粒度（タスク単位？一括？）
- アーカイブデータの同期範囲（直近N日分のみ？）
