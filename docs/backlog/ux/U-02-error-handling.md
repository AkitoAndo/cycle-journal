## ユーザーストーリー

> ユーザーとして、ネットワークエラーやサーバーエラーが発生したときに、何が起きたか分かりやすく表示してほしい。

## 受け入れ条件

- [ ] ネットワーク未接続時のオフライン表示
- [ ] API呼び出し失敗時のリトライUI
- [ ] タイムアウト時の適切なメッセージ
- [ ] 認証エラー時の再ログイン誘導

## 検討事項

- エラー表示のトーン（Cycleの世界観に合わせる）
- リトライ戦略（自動 / 手動）
- オフラインモードの範囲（ローカル機能はそのまま使える？）
