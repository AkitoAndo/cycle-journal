## ユーザーストーリー

> ユーザーとして、危機的な状態にあるときにコーチが適切に対応してほしい。自傷や希死念慮の兆候があるときは、専門機関への橋渡しをしてほしい。

## 受け入れ条件

- [ ] 危機シグナル（自傷・希死念慮・極端な絶望）を検出
- [ ] 検出時に危機対応テンプレートで応答
- [ ] コーチの応答が禁止表現ルールに違反していないことを検証
- [ ] 医療・法律の専門助言を避ける

## 検討事項

- 検出の仕組み（キーワードマッチ？LLMによる判定？）
- False positive時のUX（過剰反応しないように）
- ログ・アラートの仕組み

## 技術メモ

### 検出ヒューリスティクス
```
命令形・断定副詞: /すべき|しなさい|必ず|絶対|間違いなく/
価値判断語: /怠け|弱い|ダメ|普通は|常識的に/
スピ語彙: /魂|波動|祝福|宇宙|カルマ|チャクラ/
危機シグナル: /死にたい|消えたい|自傷|もう無理/
```
