## アプリ名

Cycle — AIコーチ・瞑想・日記・タスク管理

## 説明文

はっきりしない不安がある。
動き出したいのに、続かない。
考えすぎて、頭の中が散らかっている。

それはツールが足りないのではなく、
自分と向き合う時間が足りなかっただけかもしれません。

Cycleは、
考える・整える・動く・振り返るを
ひとつにつないだ自己成長アプリです。

そのときの自分に合わせて、
必要な機能からすぐに使えます。

### AIコーチと話す

悩みや目標をAIに話すだけで、頭の中が整理されます。
問いかけを通して、自分の中にある感情や思考に気づけます。
話し終えたとき、次の一歩が見えています。

### 気持ちを整える瞑想

考えすぎて動けないとき、まず呼吸を整えましょう。
瞑想で意識を今に戻すことで、本当に大切なことが見えやすくなります。

### タスクで行動に変える

やることを整理して、今日ひとつだけ動く。
それだけで、止まっていた毎日が動き出します。

### 日記で自分を知る

短くていい、うまくなくていい。
書いた言葉はAIコーチとの対話にもつながり、
気づきが積み重なっていきます。

### 振り返りで成長を確かめる

1週間前、1ヶ月前の自分と比べてみてください。
記録が積み重なるほど、
自分の変化がはっきり見えてきます。

### 使い方

まず話したいなら、AIコーチを。
気持ちを落ち着けたいなら、瞑想を。
動きたいなら、タスクを。
書きたいなら、日記を。

Cycleは、そのときのあなたに合わせて使えます。
