# B-01: 感情ラベル辞書

## 概要

ユーザーのジャーナルや会話から感情を読み取り、適切に分類・ラベリングするための辞書。感情を可視化し、根や土（価値観・経験）との結びつきを理解しやすくする。

---

## 分類方針

- 感情は「一次感情（基本）」と「二次感情（派生）」に整理する
- 各ラベルには「説明」「ニュアンスの幅」「関連する問いかけ例」を付与する
- **感情は否定せず、自然に受けとめる姿勢を徹底する**

---

## 一次感情カテゴリ

### 1. 喜び

| 項目 | 内容 |
|------|------|
| 二次感情例 | 安心、満足、達成感、誇り、愛情 |
| 説明 | 望みや期待が満たされているときの感情 |
| 問いかけ例 | 「その喜びの背景には、どんなことがあった？」 |

### 2. 怒り

| 項目 | 内容 |
|------|------|
| 二次感情例 | いら立ち、憤り、嫌悪、失望 |
| 説明 | 不正や不公平、期待外れに対して湧く感情 |
| 問いかけ例 | 「その怒りは、どんな大切なものを守ろうとしている？」 |

### 3. 哀しみ

| 項目 | 内容 |
|------|------|
| 二次感情例 | 喪失感、孤独、寂しさ、後悔 |
| 説明 | 失ったことや満たされないことに気づいたときの感情 |
| 問いかけ例 | 「その哀しみから、何に気づいた？」 |

### 4. 不安

| 項目 | 内容 |
|------|------|
| 二次感情例 | 恐れ、緊張、心配、焦り |
| 説明 | 未来に対する不確かさやリスクを感じたときの感情 |
| 問いかけ例 | 「その不安は、どんな未来を想像させている？」 |

### 5. 驚き

| 項目 | 内容 |
|------|------|
| 二次感情例 | 好奇心、動揺、感嘆、違和感 |
| 説明 | 予期しない出来事や気づきに出会ったときの感情 |
| 問いかけ例 | 「その驚きは、どんな意味を持ちそう？」 |

### 6. 安らぎ

| 項目 | 内容 |
|------|------|
| 二次感情例 | 落ち着き、平穏、信頼、調和 |
| 説明 | 安全で守られていると感じるときの感情 |
| 問いかけ例 | 「その安らぎを、これからどう生かせそう？」 |

---

## 実装メモ

- 入力テキストから感情語を抽出 → 上記カテゴリにマッピング
- 未知の感情語が出た場合は「類似ワード辞書」に追加
- 感情ラベルは複数同時に付与可能（例: 不安＋期待）
- 感情強度スコアを設定（弱・中・強）し、対話のトーン調整に利用

---

## まとめ

B-01は、ユーザーの言葉を「感情ラベル」として整理するための基盤辞書。大樹のように揺らがず、感情を根や土とつなげ、自然に受けとめて問いを返す。
