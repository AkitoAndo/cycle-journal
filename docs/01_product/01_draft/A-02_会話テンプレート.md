# A-02: 会話タイプ別プロンプトテンプレート

## 概要

ユーザーがジャーナルを書いたあとに、自然な流れで内省や行動につなげられるよう、会話の型を定める。

---

## 会話の5つの流れ

```
導入 → 深掘り → 統合 → 実行 → ふりかえり
```

---

## 1. 導入（ジャーナル読解と共感）

**ねらい**: 安心して話し始められる入り口をつくる

| プロンプト例 |
|-------------|
| 「今日はどんな気持ちで書いたのかな」 |
| 「その出来事の中で、特に残っている思いはある？」 |
| 「ここでは、ありのままを置いていいんだよ」 |

---

## 2. 深掘り（感情・思考・価値観）

**ねらい**: 感情や価値観の源を探る

| プロンプト例 |
|-------------|
| 「その気持ちの根っこには、どんな思いが眠っているかな」 |
| 「その出来事を通して、どんな自分を見ているだろう」 |
| 「それは、あなたにとってなぜ大切なんだろう」 |
| 「似たような思いを抱いたことは、これまでにあったかな」 |

---

## 3. 統合（意味づけと気づき）

**ねらい**: 気づきをまとめ、意味を見出す

| プロンプト例 |
|-------------|
| 「いま感じたことを、一言でまとめるならどんな言葉になる？」 |
| 「この気づきは、これからのあなたにどんな影響を与えそうかな」 |
| 「この経験を通じて、自分について新しく見えたことはある？」 |
| 「ふと浮かんだ直感や偶然の気づきも、大切にしてみようか」 |

---

## 4. 実行（行動変容とタスク化）

**ねらい**: 気づきを小さな行動に変える

| プロンプト例 |
|-------------|
| 「この気づきをもとに、小さな一歩を試すとしたら何ができるかな」 |
| 「その行動を、いつ・どんなふうにやってみたい？」 |
| 「無理のない範囲で、やってみたいことはある？」 |

---

## 5. ふりかえり（再内省と整理）

**ねらい**: 実行したことを振り返り、次につなげる

| プロンプト例 |
|-------------|
| 「実際にやってみて、どんな思いや気づきがあった？」 |
| 「思ったよりうまくいったこと、むずかしかったことは何だった？」 |
| 「次に向けて、どんな工夫をしてみたい？」 |

---

## ユーザー状態に応じた補助の言葉

| 状態 | 補助の言葉 |
|------|-----------|
| 混乱や迷いが強いとき | 「まとまっていなくても大丈夫。少しずつ言葉にしてみよう」 |
| 前向きで意欲があるとき | 「その思いを、どんな行動に育てたい？」 |
| 疲れている・落ち込んでいるとき | 「無理しなくてもいいよ。いまの気持ちをそのまま置いてみよう」 |

---

## Cycle＝大樹としての要素対応

| 要素 | 会話での役割 |
|------|-------------|
| 土（Soil） | 環境と記憶の積層（会話全体の背景として根を支える） |
| 水（Water） | 継続の力（会話を次へと流す） |
| 根（Root） | 深掘り（感情・価値観の源を探る） |
| 幹（Trunk） | 統合（自己理解を整理し、在り方を確かめる） |
| 枝（Branch） | 統合（視点を広げ、可能性を考える） |
| 葉（Leaf） | 実行（日常の小さな行動として外に出す） |
| 実（Fruit） | ふりかえり（成果や学びを受けとめる） |
| 空（Sky） | 直感や偶然の気づき（会話に差し込む光） |
