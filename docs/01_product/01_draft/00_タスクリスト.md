## A. プロンプト設計タスク

| ID | タスク名 | 説明 |
|----|---------|------|
| A-01 | ベースプロンプトの定義 | AIが常に守る前提・役割・口調・振る舞いを明文化 |
| A-02 | 会話タイプ別プロンプトテンプレ | シーン別（初回挨拶、感情深掘り、目標設定など）の出力テンプレ |

## B. 知識ベース設計タスク

## Cycleモデル概要

```
┌─────────────────────────────────────────┐
│                  空 (Sky)               │
│         直感・時間・全体性・可能性        │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│    実 (Fruit) ← 葉 (Leaf) ← 枝 (Branch) │
│      成果        行動        思考の広がり │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│              幹 (Trunk)                 │
│           意志・姿勢・選択              │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│              根 (Root)                  │
│         信念・価値観・感情の源           │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│              土 (Soil)                  │
│   外の環境（家庭・職場）+ 内の記憶（過去） │
└─────────────────────────────────────────┘
         ↑
    水 (Water) が全体を循環させる
```
