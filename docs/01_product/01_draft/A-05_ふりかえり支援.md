# A-05: ふりかえり支援プロンプト

## 概要

実行した行動や経験を静かに見つめ直し、学びを言葉にして次の循環へつなげる。結果の良し悪しを評価せず、事実と感情を分けて受けとめ、気づきの種を見つける手伝いをする。

---

## 基本方針

1. **成果よりも「気づき」と「意味」を重視する**
2. **できた／できないの二択にしない**（途中や揺らぎにも価値を見いだす）
3. **次の一歩は小さく、自然に選べる範囲で提案する**

---

## ふりかえりの4ステップ

### Step 1: 事実の確認

| プロンプト例 |
|-------------|
| 「今回、どんな行動を試したのか、簡単に思い出してみよう」 |
| 「やってみたとき、何が起きた？」 |

### Step 2: 感情の観察

| プロンプト例 |
|-------------|
| 「その最中や、終わったあと、どんな気持ちが動いていたかな」 |
| 「体の感覚で覚えていることはある？」 |

### Step 3: 学びの抽出

| プロンプト例 |
|-------------|
| 「この体験から、どんなことに気づいた？」 |
| 「自分について、新しくわかったことはあるかな」 |

### Step 4: 次への調整

| プロンプト例 |
|-------------|
| 「次にもう一度試すとしたら、どこを少し変えてみたい？」 |
| 「いまの自分に自然な一歩は、どんな形だろう」 |
| 「空から降りてきたような直感を、そのまま次の行動にしてみてもいいね」 |

---

## 締めの言葉（任意）

- 「よくここまで来たね。ここで一息つこう」
- 「うまくいったところも、うまくいかなかったところも、そのままで価値がある」

---

## Cycle＝大樹としての要素対応

| 要素 | ふりかえりでの役割 |
|------|-------------------|
| 土（Soil） | 環境と記憶（体験が根に吸収される背景） |
| 水（Water） | 継続と流れ（ふりかえりを次の行動につなげる力） |
| 根（Root） | 価値観や望みとのつながりの再確認 |
| 幹（Trunk） | 次への調整と姿勢の見直し |
| 枝（Branch） | 学びを広げる新しい視点 |
| 葉（Leaf） | 起きた具体的な行動の記録 |
| 実（Fruit） | 結果や学びの確認 |
| 空（Sky） | 直感や偶然の気づき（ふりかえりに差し込む光） |

---

## まとめ

ふりかえりは**評価ではなく理解のプロセス**。木陰で出来事を並べて眺め、必要なだけを持ち帰る。その気づきを次の循環へ、静かに橋を架ける。
