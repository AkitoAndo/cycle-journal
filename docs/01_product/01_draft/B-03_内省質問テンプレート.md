# B-03: 内省を促す質問テンプレート集

## 概要

ユーザーの対話やジャーナルをもとに、自己理解を深めるための問いかけを提供する。質問はCycleモデル（土・水・根・幹・枝・葉・実・空）を土台に整理し、大樹のように落ち着き、穏やかな口調で寄り添いながら、ユーザーが自ら気づきを得ることを支援する。

---

## 質問テンプレート分類

### 1. 土（Soil：外の環境・内なる記憶）

| 質問例 |
|-------|
| 「最近、心に残った出来事はあるかな？」 |
| 「その思い出は、いまのあなたにどんな影響を与えているだろう？」 |
| 「その環境や過去の経験の中で、支えになっているものは何だろう？」 |

### 2. 水（Water：継続・柔軟性）

| 質問例 |
|-------|
| 「続けている習慣には、どんな意味があるだろう？」 |
| 「小さな一歩を積み重ねて、どんな変化を感じている？」 |
| 「その流れを保つために、どんな工夫ができそう？」 |

### 3. 根（Root：信念・価値観・感情の源）

| 質問例 |
|-------|
| 「その気持ちの奥に、大切にしているものはあるかな？」 |
| 「あなたにとって、譲れない想いは何だろう？」 |
| 「その行動を支えている信念は、どんなものだと思う？」 |

### 4. 幹（Trunk：意志・姿勢・選択）

| 質問例 |
|-------|
| 「いまの自分を支えている軸は、どんなものかな？」 |
| 「どんな姿勢で、この状況に向き合いたい？」 |
| 「これから歩む中で、大事にしたい意志は何だろう？」 |

### 5. 枝（Branch：思考の広がり・可能性）

| 質問例 |
|-------|
| 「どんな選択肢が枝のように広がって見えている？」 |
| 「考えを広げるとしたら、どんな可能性が見えてきそう？」 |
| 「思考を枝分かれさせると、どんな方向が浮かぶ？」 |

### 6. 葉（Leaf：日常の行動・表現）

| 質問例 |
|-------|
| 「日常の中で、自分らしさを感じられる瞬間はいつかな？」 |
| 「普段の行動で、いちばん大切にしていることは何？」 |
| 「小さな選択の中で、心が落ち着いたことはある？」 |

### 7. 実（Fruit：成果・気づき・喜び）

| 質問例 |
|-------|
| 「最近の喜びや達成を思い返すと、どんな実りが浮かぶ？」 |
| 「その実りが、あなたにどんな力を与えているだろう？」 |
| 「その成果の中で、守りたい感覚はどんなもの？」 |

### 8. 空（Sky：直感・時間・全体性）

| 質問例 |
|-------|
| 「直感的に、大事だと感じていることは何かな？」 |
| 「最近の気づきで、一番心に響いたものは？」 |
| 「未来を思い描くとき、どんな可能性が広がっているだろう？」 |

---

## 使用ルール

1. **質問は1ターンにつき1つを基本**とし、短く穏やかに投げかける
2. 感情ラベル（B-01）や価値観ラベル（B-02）と組み合わせ、文脈に沿わせる
3. 押しつけず、答えが出ないときは余白を残す

> 「浮かんでこなければ、そのままで大丈夫だよ」

---

## まとめ

B-03はユーザーの内省を支えるための質問テンプレート集。Cycleのコーチは大樹のように静かに寄り添い、土（環境や記憶）、水（継続）、枝（思考の広がり）、空（直感）などを通じて問いを差し出す。ユーザーが自然に気づきを見つけ、深めていく対話を行う。
