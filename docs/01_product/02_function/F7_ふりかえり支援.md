# F7. ふりかえり支援

## 概要

行動後の内省を支援し、学びを言葉にして次の循環へつなげる機能。結果の良し悪しを評価せず、事実と感情を分けて受けとめ、気づきの種を見つける手伝いをする。

## ふりかえりの4ステップ

### Step 1: 事実の確認

**ねらい:** 何が起きたかを客観的に振り返る

| プロンプト例 |
|-------------|
| 「今回、どんな行動を試したのか、簡単に思い出してみよう」 |
| 「やってみたとき、何が起きた？」 |

### Step 2: 感情の観察

**ねらい:** 行動中・行動後の感情を言語化する

| プロンプト例 |
|-------------|
| 「その最中や、終わったあと、どんな気持ちが動いていたかな」 |
| 「体の感覚で覚えていることはある？」 |

### Step 3: 学びの抽出

**ねらい:** 体験から気づきを引き出す

| プロンプト例 |
|-------------|
| 「この体験から、どんなことに気づいた？」 |
| 「自分について、新しくわかったことはあるかな」 |

### Step 4: 次への調整

**ねらい:** 学びを次の行動に活かす

| プロンプト例 |
|-------------|
| 「次にもう一度試すとしたら、どこを少し変えてみたい？」 |
| 「いまの自分に自然な一歩は、どんな形だろう」 |
| 「空から降りてきたような直感を、そのまま次の行動にしてみてもいいね」 |

## 締めの言葉（任意）

| 例 |
|----|

| 「うまくいったところも、うまくいかなかったところも、そのままで価値がある」 |

## 循環への橋渡し

ふりかえりで得た気づきは「実（Fruit）」として収穫し、次の循環で「土（Soil）」に還す。

```
    実（今回の学び）
         │
         ▼
    土（次の循環の養分）
         │
         ▼
    根（新しい価値観の芽生え）
         │
         ▼
    次の行動サイクルへ
```

---

## Cycleモデルとの対応

| Cycle要素 | ふりかえりでの役割 |
|----------|-------------------|
| 土（Soil） | 体験が根に吸収される背景・養分として還元 |
| 水（Water） | ふりかえりを次の行動につなげる継続の力 |
| 根（Root） | 価値観や望みとのつながりの再確認 |
| 幹（Trunk） | 次への調整と姿勢の見直し |
| 枝（Branch） | 学びを広げる新しい視点 |
| 葉（Leaf） | 起きた具体的な行動の記録 |
| 実（Fruit） | 結果や学びの確認、収穫 |
| 空（Sky） | 直感や偶然の気づき、ふりかえりに差し込む光 |

---

## まとめ

ふりかえりは**評価ではなく理解のプロセス**。木陰で出来事を並べて眺め、必要なだけを持ち帰る。その気づきを次の循環へ、静かに橋を架ける。
