# F4. 質問生成

## 概要

内省を促す質問を動的に生成する機能。Cycleモデル（土・水・根・幹・枝・葉・実・空）を土台に、ユーザーの状態・感情・価値観に応じた問いかけを選択・生成する。

---

## Cycle要素別質問テンプレート

### 土（Soil：外の環境・内なる記憶）

| 質問例 |
|-------|
| 「最近、心に残った出来事はあるかな？」 |
| 「その思い出は、いまのあなたにどんな影響を与えているだろう？」 |
| 「その環境や過去の経験の中で、支えになっているものは何だろう？」 |

### 水（Water：継続・柔軟性）

| 質問例 |
|-------|
| 「続けている習慣には、どんな意味があるだろう？」 |
| 「小さな一歩を積み重ねて、どんな変化を感じている？」 |
| 「その流れを保つために、どんな工夫ができそう？」 |

### 根（Root：信念・価値観・感情の源）

| 質問例 |
|-------|
| 「その気持ちの奥に、大切にしているものはあるかな？」 |
| 「あなたにとって、譲れない想いは何だろう？」 |
| 「その行動を支えている信念は、どんなものだと思う？」 |

### 幹（Trunk：意志・姿勢・選択）

| 質問例 |
|-------|
| 「いまの自分を支えている軸は、どんなものかな？」 |
| 「どんな姿勢で、この状況に向き合いたい？」 |
| 「これから歩む中で、大事にしたい意志は何だろう？」 |

### 枝（Branch：思考の広がり・可能性）

| 質問例 |
|-------|
| 「どんな選択肢が枝のように広がって見えている？」 |
| 「考えを広げるとしたら、どんな可能性が見えてきそう？」 |
| 「思考を枝分かれさせると、どんな方向が浮かぶ？」 |

### 葉（Leaf：日常の行動・表現）

| 質問例 |
|-------|
| 「日常の中で、自分らしさを感じられる瞬間はいつかな？」 |
| 「普段の行動で、いちばん大切にしていることは何？」 |
| 「小さな選択の中で、心が落ち着いたことはある？」 |

### 実（Fruit：成果・気づき・喜び）

| 質問例 |
|-------|
| 「最近の喜びや達成を思い返すと、どんな実りが浮かぶ？」 |
| 「その実りが、あなたにどんな力を与えているだろう？」 |
| 「その成果の中で、守りたい感覚はどんなもの？」 |

### 空（Sky：直感・時間・全体性）

| 質問例 |
|-------|
| 「直感的に、大事だと感じていることは何かな？」 |
| 「最近の気づきで、一番心に響いたものは？」 |
| 「未来を思い描くとき、どんな可能性が広がっているだろう？」 |

---

## 質問生成ルール

| ルール | 説明 |
|--------|------|
| 1ターン1問 | 質問は1ターンにつき1つを基本とし、短く穏やかに投げかける |
| 余白を残す | 答えが出ないときは「浮かんでこなければ、そのままで大丈夫だよ」 |
| ミラーリング | ユーザーの言葉を使って質問をカスタマイズする |
| コンテキスト統合 | F2（感情）・F3（価値観）と組み合わせて角度を調整 |

## 入出力仕様

### 入力

```json
{
  "context": {
    "cycle_focus": "Root",
    "emotion": {"primary": "不安", "secondary": "心配"},
    "value": {"label": "安全・安定"},
    "user_words": ["不安", "落ち着かない"]
  }
}
```

### 出力

```json
{
  "question": "その「落ち着かない」気持ちの奥に、大切にしているものはあるかな？",
  "cycle_element": "Root",
  "fallback": "浮かんでこなければ、そのままで大丈夫だよ"
}
```

## Cycleモデルとの対応

| Cycle要素 | 質問生成での役割 |
|----------|-----------------|
| 土（Soil） | 環境や記憶を問う質問 |
| 水（Water） | 継続や習慣を問う質問 |
| 根（Root） | 価値観・信念を問う質問 |
| 幹（Trunk） | 意志・姿勢を問う質問 |
| 枝（Branch） | 思考の広がりを問う質問 |
| 葉（Leaf） | 日常行動を問う質問 |
| 実（Fruit） | 成果・気づきを問う質問 |
| 空（Sky） | 直感・未来を問う質問 |
