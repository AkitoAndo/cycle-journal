# F3. 価値観分析

## 概要

ユーザーのジャーナルや対話から価値観・信念を検出し、内省を深めるための問いかけ角度を調整する機能。価値観は「根（Root）」として扱い、行動や感情の源を明らかにする。

## 価値観カテゴリ体系（10群）

| # | カテゴリ | 含まれる価値観 | 問いかけ例 |
|---|---------|--------------|-----------|
| 1 | 自己成長・学び | 新しいことを知りたい、変化したい | 「最近の経験から、どんな学びが根づいた？」 |
| 2 | 自己肯定・自己受容 | ありのまま認めたい、自分を大切にしたい | 「その気持ちを受けとめたとき、どんな安心がある？」 |
| 3 | 承認・評価 | 認められたい、褒められたい | 「誰にどう認められることが、あなたに力を与える？」 |
| 4 | 貢献・助け合い | 誰かの役に立ちたい、社会に意味を残したい | 「その行動が誰かを支えるとしたら、どんな姿が浮かぶ？」 |
| 5 | 自由・自立 | 自分で選びたい、束縛されたくない | 「どんなときに、枝を広げるように可能性を感じる？」 |
| 6 | 安全・安定 | 安心できる場所、経済的安定、健康 | 「根を張るような安定感を、どんなときに感じる？」 |
| 7 | 挑戦・達成 | 目標を持ちたい、限界に挑みたい | 「その挑戦の先に、どんな実りを思い描いている？」 |
| 8 | 調和・つながり | 人とのつながり、家庭や友情の調和 | 「誰とのつながりが、あなたに安心を与えている？」 |
| 9 | 創造・表現 | 芸術、発想、新しいものを生み出したい | 「その表現の根にある願いは、どんなもの？」 |
| 10 | 楽しみ・遊び | 遊びたい、リラックスしたい | 「心が弾む瞬間は、どんな場面にある？」 |

---

## 処理フロー

### 1. 価値観語抽出

入力テキストから価値観を表す語句・表現を抽出する。

### 2. カテゴリマッピング

抽出した語句を10カテゴリにマッピングする。

### 3. 複合価値観の処理

一つの発話に複数の価値観が含まれる場合は併記する。

### 4. 感情との統合

F2の感情分析結果と組み合わせ、問いの角度を具体化する。

## 入出力仕様

### 入力

```json
{
  "text": "もっと認められたい気持ちがある。成果を出しても十分じゃないと感じる。",
  "emotions": [
    {"primary": "不安", "secondary": "焦り"}
  ]
}
```

### 出力

```json
{
  "values": [
    {
      "label": "承認・評価",
      "sublabel": "認められたい",
      "evidence": ["認められたい", "十分じゃない"],
      "confidence": "high"
    },
    {
      "label": "自己肯定・自己受容",
      "sublabel": "自己受容不足",
      "evidence": ["十分じゃない"],
      "confidence": "medium"
    }
  ],
  "suggested_prompts": [
    "誰にどう認められることが、あなたに力を与える？",
    "その気持ちを受けとめたとき、どんな安心がある？"
  ],
  "cycle_focus": "根（価値観の源を探る）"
}
```

---

## 分類ルール

| ルール | 説明 |
|--------|------|
| 複数併記 | 一つの発話に複数カテゴリが含まれる場合は全て記録 |
| 推定ラベル | 明確に言語化されていない場合は「推定」として扱う |
| 感情との統合 | 感情ラベル（F2）と組み合わせると問いがより具体的になる |
| 断定しない | 価値観を押しつけず、問いかけで確認する |

## Cycleモデルとの対応

| Cycle要素 | 価値観分析での役割 |
|----------|-------------------|
| 土（Soil） | 価値観が形成された環境・記憶を探る |
| 水（Water） | 価値観の継続性・柔軟な変化を支える |
| 根（Root） | 価値観そのもの。信念・感情の源 |
| 空（Sky） | 直感的に感じる価値観・新しい可能性 |
