# F6. 行動提案

## 概要

ユーザーの気づきを小さな行動タスクに変換し、日常に根づかせるための提案機能。行動は強制ではなく「実りを迎えるための一歩」として、柔らかく提示する。

## 行動支援の6原則

| # | 原則 | Cycle要素 | 説明 |
|---|------|----------|------|
| 1 | 行動は小さく、具体的に | 葉 | 軽やかに実行できるサイズに |
| 2 | 価値観に結びつける | 根 | 持続可能性を高める |
| 3 | 成果より試みを大切に | 実 | 結果の有無を問わない |
| 4 | 環境を耕す視点 | 土 | 小さな変化の積み重ね |
| 5 | 無理なく続けられる工夫 | 水 | 継続と流れを重視 |
| 6 | 柔軟に選べる余白 | 空 | 直感を尊重する |

---

## 行動変容モデルとの対応

### SMART目標

| 要素 | 意味 | Cycle対応 |
|------|------|----------|
| Specific | 具体的 | 葉 |
| Measurable | 測定可能 | 実 |
| Achievable | 達成可能 | 幹 |
| Relevant | 関連性 | 根 |
| Time-bound | 期限設定 | 空 |

### BJ Fogg 行動モデル

| 要素 | 意味 | Cycle対応 |
|------|------|----------|
| 動機 | 行動の理由 | 根（価値観・感情） |
| 能力 | 実行力 | 幹（意志・姿勢） |
| きっかけ | トリガー | 空（直感・外部信号） |

### Tiny Habits

小さな習慣を積み重ねて定着させる → **水**（一滴の積み重ね）

### ナッジ理論

環境デザインで無意識に後押し → **土**（環境を耕す）

## プロンプト例

| 状況 | プロンプト例 |
|------|-------------|
| 気づきから行動へ | 「この気づきを、今日の小さな一歩にするとしたら、どんな形になりそう？」 |
| 持続性の確認 | 「その一歩を根づかせるために、どんな工夫ができるだろう」 |
| 具体化の促し | 「できることを、ほんの少しでいいから具体的に描いてみよう」 |
| 感情の予測 | 「やってみた後に、どんな気持ちになりそうか、想像してみようか」 |
| 直感の活用 | 「ふと浮かんだ直感を、試しに行動にしてみてもいいね」 |

```json
  },
```

```json
    "trigger": "起床後すぐ"

  "fallback": "無理のない範囲で、やってみたいことがあれば教えてね"

```

## 実装ルール

| ルール | 説明 |
|--------|------|
| 小さな一歩に限定 | 大きな目標は分解して提案 |
| 価値観と結びつける | 根（価値観）や土（環境）との関連を示す |
| SMART準拠 | 具体的・測定可能・達成可能・関連・期限を意識 |
| 柔らかく問いかける | 「〜してみる？」の形式で選択肢を残す |
| 押しつけない | ユーザーが選べる余白を確保 |

---

## Cycleモデルとの対応

| Cycle要素 | 行動提案での役割 |
|----------|-----------------|
| 土（Soil） | 環境を整え、行動が育つ基盤をつくる |
| 水（Water） | 継続を支え、習慣として流れをつくる |
| 根（Root） | 価値観に結びつけ、動機を明確にする |
| 幹（Trunk） | 意志を確認し、実行力を支える |
| 枝（Branch） | 選択肢を広げ、柔軟性を持たせる |
| 葉（Leaf） | 具体的で小さな日常行動として表現 |
| 実（Fruit） | 行動の結果・気づきを受けとめる |
| 空（Sky） | 直感やきっかけを行動に取り入れる |
