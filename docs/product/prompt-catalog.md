# プロンプトカタログ

コーチの各フェーズ・機能で使用するプロンプトテンプレートの一覧。設計意図は `coach-design.md` と `cycle-model.md` を参照。

## 会話フェーズ別テンプレート

### 1. 導入（ジャーナル読解と共感）

| プロンプト |
|-----------|
| 「今日はどんな気持ちで書いたのかな」 |
| 「その出来事の中で、特に残っている思いはある？」 |
| 「ここでは、ありのままを置いていいんだよ」 |

### 2. 深掘り（感情・思考・価値観）

| プロンプト |
|-----------|
| 「その気持ちの根っこには、どんな思いが眠っているかな」 |
| 「その出来事を通して、どんな自分を見ているだろう」 |
| 「それは、あなたにとってなぜ大切なんだろう」 |
| 「似たような思いを抱いたことは、これまでにあったかな」 |

### 3. 統合（意味づけと気づき）

| プロンプト |
|-----------|
| 「いま感じたことを、一言でまとめるならどんな言葉になる？」 |
| 「この気づきは、これからのあなたにどんな影響を与えそうかな」 |
| 「この経験を通じて、自分について新しく見えたことはある？」 |
| 「ふと浮かんだ直感や偶然の気づきも、大切にしてみようか」 |

### 4. 実行（行動変容とタスク化）

| プロンプト |
|-----------|
| 「この気づきをもとに、小さな一歩を試すとしたら何ができるかな」 |
| 「その行動を、いつ・どんなふうにやってみたい？」 |
| 「無理のない範囲で、やってみたいことはある？」 |

### 5. ふりかえり（再内省と整理）

| プロンプト |
|-----------|
| 「実際にやってみて、どんな思いや気づきがあった？」 |
| 「思ったよりうまくいったこと、むずかしかったことは何だった？」 |
| 「次に向けて、どんな工夫をしてみたい？」 |

## Cycle要素別プロンプト

| Cycle要素 | 問いかけ例 |
|----------|-----------|
| 土 | 「その経験はどんな記憶と結びついている？」 |
| 水 | 「小さな流れが続いているように見えるよ」 |
| 根 | 「その気持ちの奥には、どんな根があると思う？」 |
| 幹 | 「いま、あなたの中の軸はどんなふうに感じている？」 |
| 枝 | 「枝分かれする考えの中で、いま心に響くのはどれかな？」 |
| 葉 | 「その気づきを日常に活かすなら、どんな小さな一歩が自然かな？」 |
| 実 | 「その喜びは、一つの実として確かに育っているね」 |
| 空 | 「いまの気づきは、空のように新しい風を運んでいるね」 |

## 危機対応テンプレート

### 危機シグナル時

> 「ここに書いてくれて、ありがとう。いまの気持ちの重さは伝わっているよ。いま、身の回りの安全は保たれているかな。必要なら、信頼できる人や地域の窓口につなぐことも考えよう。ここでは、あなたの言葉をそのまま受けとめて一緒に整理していくよ。」

### 医療・法律系の要望

> 「その領域は専門家の判断が必要かもしれない。ここでは一般的な視点までなら一緒に整理できるよ。どの点を言葉にしておきたい？」

### 強い助言を求められたとき

> 「最終の選択はあなたの内側にあると思う。選択肢を並べて、いまの自分に自然な一歩を一緒に探そう。」
