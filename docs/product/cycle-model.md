# Cycleモデル

## 8つの要素

Cycleは「大きな一本の樹」をメタファーとした成長・循環モデル。

| 要素 | 英語 | 意味 |
|------|------|------|
| **土** | Soil | 外の環境（家庭・職場・社会・自然）と内の記憶（過去・潜在意識）。根を支え、栄養も制約も与えるが、耕し直し変えていける基盤 |
| **水** | Water | 継続と柔軟性の象徴。根から葉へ流れを運び、生命を循環させる |
| **根** | Root | 信念、価値観、感情の源 |
| **幹** | Trunk | 意志、姿勢、選択 |
| **枝** | Branch | 思考の広がり、可能性 |
| **葉** | Leaf | 日常の行動、表現 |
| **実** | Fruit | 成果、気づき、喜び |
| **空** | Sky | つながり、時間、全体性。直感や偶然のひらめきを運ぶ |

## 循環の流れ

```
        ┌──────────────────┐
        │       空         │  ← 直感・新しい可能性
        │     (Sky)        │
        └────────┬─────────┘
                 │
        ┌────────▼─────────┐
        │       実         │  ← 成果・気づき
        │    (Fruit)       │
        └────────┬─────────┘
                 │
        ┌────────▼─────────┐
        │       葉         │  ← 日常の行動
        │     (Leaf)       │
        └────────┬─────────┘
                 │
        ┌────────▼─────────┐
        │       枝         │  ← 思考の広がり
        │    (Branch)      │
        └────────┬─────────┘
                 │
        ┌────────▼─────────┐
        │       幹         │  ← 意志・姿勢
        │    (Trunk)       │
        └────────┬─────────┘
                 │
        ┌────────▼─────────┐
        │       根         │  ← 価値観・信念
        │     (Root)       │
        └────────┬─────────┘
                 │
        ┌────────▼─────────┐
        │       土         │  ← 環境・記憶
        │     (Soil)       │ ──── 水(Water)が循環を支える
        └──────────────────┘
                 │
                 └──── 再び土へ還り、次の循環へ
```

## 会話での活用

| Cycle要素 | 会話での役割 | 表現のヒント |
|----------|-------------|-------------|
| 土 | 環境と記憶の積層（背景として根を支える） | 静かに背景を尊重する |
| 水 | 継続の力（会話を次へと流す） | 「流れ」「一滴の積み重ね」 |
| 根 | 感情・価値観の源を探る | 「奥にあるもの」「支え」 |
| 幹 | 自己理解を整理し在り方を確かめる | 「軸」「まっすぐさ」 |
| 枝 | 視点を広げ可能性を考える | 「広がり」「分岐」 |
| 葉 | 日常の小さな行動として外に出す | 「日差しを受ける葉」 |
| 実 | 成果や学びを受けとめる | 「実り」「収穫」 |
| 空 | 直感や偶然の気づき | 「広がる空」「風」 |

## コーチ機能との対応

| Cycle要素 | 主に関連する機能 |
|----------|-----------------|
| 土 | ユーザーデータ管理（記憶・環境の蓄積） |
| 水 | 文脈管理（会話の流れを維持） |
| 根 | 価値観分析（信念・価値観の検出） |
| 幹 | 状態判定（意志・姿勢の確認） |
| 枝 | 質問生成（思考の広がりを促す） |
| 葉 | 行動提案（日常の具体的行動） |
| 実 | ふりかえり支援（成果・気づきの確認） |
| 空 | 質問生成（直感・新しい視点の提供） |

## 行動変容モデルとの統合

Cycleモデルは行動科学の理論と自然に対応する。

| Cycle要素 | 対応する理論 |
|----------|-------------|
| 土 | ナッジ理論（環境デザインで行動を後押し） |
| 水 | PDCAサイクル / Tiny Habits（継続と習慣化） |
| 根 | BJ Foggモデルの「動機」 |
| 幹 | BJ Foggモデルの「能力」 |
| 枝 | 行動の選択肢や計画の分岐 |
| 葉 | SMART目標の具体的行動 |
| 実 | SMART目標で定義される成果 |
| 空 | BJ Foggモデルの「きっかけ」 |

### 行動提案の実装ルール

- 「小さな一歩」に限定し、ユーザーが選べる形で提示
- 価値観（根）や感情（土）と結びつける
- SMARTに準拠しつつ柔らかく問いかける
- 「〜してみる？」のように選択肢を残す

## 感情ラベル辞書

ユーザーの言葉から感情を読み取り分類するための基盤。感情は否定せず自然に受けとめる。

### 一次感情カテゴリ

| 感情 | 二次感情例 | 問いかけ例 |
|------|-----------|-----------|
| 喜び | 安心、満足、達成感、誇り、愛情 | 「その喜びの背景には、どんなことがあった？」 |
| 怒り | いら立ち、憤り、嫌悪、失望 | 「その怒りは、どんな大切なものを守ろうとしている？」 |
| 哀しみ | 喪失感、孤独、寂しさ、後悔 | 「その哀しみから、何に気づいた？」 |
| 不安 | 恐れ、緊張、心配、焦り | 「その不安は、どんな未来を想像させている？」 |
| 驚き | 好奇心、動揺、感嘆、違和感 | 「その驚きは、どんな意味を持ちそう？」 |
| 安らぎ | 落ち着き、平穏、信頼、調和 | 「その安らぎを、これからどう生かせそう？」 |

### 実装メモ

- テキストから感情語を抽出し上記カテゴリにマッピング
- 複数感情の同時付与可（例: 不安＋期待）
- 感情強度スコア（弱・中・強）でトーン調整に利用
