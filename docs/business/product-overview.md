# プロダクト概要

## ビジョン

Cycle Journal は、**AIコーチと一緒に日々の気づきを深める**ジャーナリングアプリです。

ユーザーが日記を書き、AIコーチとの対話を通じて自己理解を深め、行動タスクの実行と振り返りによって成長のサイクルを回していくことを目的としています。

## Cycle モデル（大樹メタファー）

プロダクトの中核をなす概念モデルです。人間の成長と内省を「大樹」の要素で表現します。

| 要素 | メタファー | 意味 |
|------|-----------|------|
| 🌍 土（Soil） | 外部環境と内的記憶 | あなたを取り巻く環境、過去の経験 |
| 💧 水（Water） | 連続性・柔軟性 | 流れるエネルギー、変化への適応力 |
| 🌱 根（Root） | 信念・価値観 | 感情の基盤、深層にある思い |
| 🪵 幹（Trunk） | 意志・姿勢 | 選択と決断の軸 |
| 🌿 枝（Branch） | 広がる思考 | 可能性の探索、新しい視点 |
| 🍃 葉（Leaf） | 日々の行動 | 具体的なアクション、表現 |
| 🍎 実（Fruit） | 成果・気づき | 結果、喜び、学び |
| ☁️ 空（Sky） | つながり・時間 | 全体を俯瞰する視点 |

## ターゲットユーザー

- 日々の内省を習慣化したい人
- 自己理解を深めたいが、一人では言語化が難しい人
- コーチングに興味があるが、対人でのハードルが高い人
- 行動と振り返りのサイクルを回して成長を実感したい人

## コアバリュー

1. **安心・安全な対話空間** — ジャッジメントのない AI コーチが寄り添う
2. **自己発見の促進** — 答えを与えるのではなく、問いを通じて気づきを引き出す
3. **行動と振り返りの循環** — タスク実行 → 4ステップ振り返り → 次の行動へ
4. **プライバシーファースト** — 日記はデバイスローカル保存、クラウドは最小限

### 日記（Diary）
- デバイスローカルに保存（プライバシー重視）
- タグによる分類・検索
- 週間カレンダービュー
- コーチとの対話に文脈として共有可能

### AIコーチ（Coach）
- AWS Bedrock（Claude 3 Haiku）を活用
- Cycle モデルに基づいた共感的な問いかけ
- 1〜3文の簡潔な応答
- セッション管理と履歴保存

### タスク（Tasks）
- コーチとの対話から生まれる行動タスク
- 期限管理とステータス管理
- **4ステップ振り返り**:
  1. 事実の確認 — 何をした？
  2. 感情の観察 — どう感じた？
  3. 学びの抽出 — 何に気づいた？
  4. 次への調整 — 次はどうする？
