# プライバシーポリシー

## Cycle Journal について

Cycle Journal は AI コーチングと日記機能を提供する iOS アプリケーションです。

## 収集する情報

このアプリは以下の方針でユーザーの情報を扱います：

### 認証情報
- **Sign in with Apple**: Apple ID によるサインイン時に、Apple から提供されるユーザー ID とメールアドレス（リレーアドレス可）を取得します
- 認証トークンはデバイスの Keychain にセキュアに保存されます

### ローカルに保存されるデータ
- **日記データ**: ユーザーが入力した日記データはすべてデバイス内（UserDefaults）にのみ保存されます
- **ユーザー設定**: 通知設定などはデバイス内に保存されます

### クラウドに送信されるデータ
- **コーチとの対話内容**: AI コーチとの対話メッセージはサーバーに送信・保存されます
- **タスクデータ**: 行動タスクおよび振り返り内容はサーバーに保存されます
- **日記コンテキスト（任意）**: ユーザーが明示的に選択した場合のみ、日記内容がコーチとの対話のコンテキストとしてサーバーに送信されます

## AI サービスの利用

- コーチ機能では AWS Bedrock（Claude）を利用しています
- ユーザーのメッセージは AI の応答生成のために処理されます
- AI サービスプロバイダーの利用規約・プライバシーポリシーも適用されます

## データの保存と管理

| データ種別 | 保存場所 | 送信 |
|-----------|---------|------|
| 日記 | デバイスのみ | なし |
| コーチ対話 | サーバー | あり |
| タスク・振り返り | サーバー | あり |
| 認証トークン | Keychain | 検証時のみ |
| ユーザー設定 | デバイスのみ | なし |

- アプリを削除すると、デバイス内のデータは完全に削除されます
- サーバー上のデータの削除はデータエクスポート画面から申請できます

## 第三者との情報共有

このアプリは第三者とユーザーの個人情報を共有することはありません。ただし、AI コーチ機能の提供に必要な範囲で AWS のサービスを利用しています。

## 分析・追跡

このアプリには以下が含まれていません：
- アナリティクス（使用状況の追跡）
- 広告
- クラッシュレポートの外部送信

## お子様のプライバシー

このアプリは 13 歳未満のお子様から故意に個人情報を収集することはありません。
