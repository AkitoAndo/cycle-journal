## コアストーリー

### 1. 日記を書く

> ユーザーとして、日々の出来事や気持ちを自由に記録したい。それにより、自分の内面を言語化する習慣をつけたい。

**受け入れ条件:**
- テキストで自由に日記を入力できる
- タグを付けて分類できる
- 日記はデバイス内のみに保存される（プライバシー確保）
- 過去の日記を検索・閲覧できる
- 週間カレンダーから日記を探せる

### 2. AIコーチと対話する

> ユーザーとして、AIコーチに話を聞いてもらいたい。答えを教えてもらうのではなく、問いかけを通じて自分で気づきを得たい。

**受け入れ条件:**
- コーチにメッセージを送ると、共感的な応答が返ってくる
- コーチは1〜3文の簡潔な応答をする
- 答えを押し付けず、オープンエンドな問いかけをしてくれる
- 対話はセッション単位で保存される
- 過去のセッション履歴を振り返れる

### 3. 日記をもとにコーチと対話する

> ユーザーとして、書いた日記の内容をコーチに共有して、より深い対話をしたい。

**受け入れ条件:**
- 日記一覧からコーチへの共有ができる
- コーチは日記の文脈を踏まえた応答をしてくれる
- どの日記を共有するかはユーザーが選択できる

### 4. タスクを作成・管理する

> ユーザーとして、コーチとの対話から生まれた「やってみよう」を行動タスクとして記録したい。

**受け入れ条件:**
- タスクにタイトル・説明・期限を設定できる
- 未完了・完了のステータスを管理できる
- 期限超過のタスクが視覚的にわかる
- タスクの編集・削除ができる

### 5. タスクを振り返る

> ユーザーとして、完了したタスクについて構造化された振り返りをしたい。それにより、行動から学びを得て次に活かしたい。

**受け入れ条件:**
- タスク完了時に振り返りフローが起動する
- 4ステップで段階的に振り返れる（事実→感情→学び→次のアクション）
- 各ステップの入力がガイド付きで行える
- 振り返りの内容は後から閲覧・編集できる
- 完了時にお祝い表示がある

### 6. 認証する

> ユーザーとして、Apple ID で簡単にサインインしたい。個人情報は最小限で済ませたい。

**受け入れ条件:**
- Sign in with Apple でワンタップサインイン
- メールアドレスは Apple のリレーアドレスで匿名化可能
- 認証トークンはセキュアに保存される
- サインアウトが簡単にできる

## ユーザーフロー

### 初回利用フロー

```
アプリ起動
    ↓
サインイン画面
    ↓
Sign in with Apple
    ↓
メイン画面（日記タブ）
    ↓
初めての日記を書く
    ↓
コーチと話してみる（任意）
```

### 日常利用フロー（成長のサイクル）

```
日記を書く → コーチと対話 → タスクを作成 → タスクを実行 → 振り返り → 日記を書く → ...
     ↑                                                                    │
     └────────────────────── 成長のサイクル ──────────────────────────────────┘
```

### 振り返りフロー

```
タスク完了をタップ
    ↓
ステップ1: 事実の確認（何をした？）
    ↓
ステップ2: 感情の観察（どう感じた？）
    ↓
ステップ3: 学びの抽出（何に気づいた？）
    ↓
ステップ4: 次への調整（次はどうする？）
    ↓
完了セレブレーション 🎉
```
